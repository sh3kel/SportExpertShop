SportExpertShop - папка, в которой хранится проект
    SportExpert - корневая директория проекта
        Core - директория основного приложение нашего проекта
             migrations - директория с файлами миграции базы данных проекта
             static - директория для хранения статических файлов(CSS-файлы, JS-скрипты)
             templatetags - директория для хранения Python-файлов с тегами шаблонов
             admin.py - Python файл для подключения моделей базы данных к админ-панели и их настройки
             apps.py - Python файл с конфигурацией приложения Core
             forms.py - Python файл с описанием форм, подключенных в нашем проекте(форма авторизации, регистрации, оплаты)
             models.py - Python файл с описанием моделей в нашей базе данных, их функций и методов
             tests.py - Python файл для написания unit-тестов
             urls.py Python файл для создания ссылок в нашем приложении
             views.py Python файл для описания представления страниц и функций перехода между ними.
        Media - директория с медиа-файлами(картинки, видео и пр.) нашего проекта
        Sportexpert - директория с конфигурацией, настройками нашего проекта и ядра Django.
            settings.py - настройки нашего проекта
            urls.py - подключение ссылок к ядру Django
            asgi.py
            wsgi.py - файлы веб-технологий подключения клиента и сервера
        templates - директория с HTML шаблонами нашего проекта
        db.sqlite3 - база данных проекта
        manage.py - файл управления Django-проектом.
